#GPractice

###Installation
1. `go install -v ...`
2. - `gpractice` to run CLI
   - `gpractice-web` to run web app, default URL: `http://localhost:3000/app`
3. to see available arguments, run needed command with flag `-help`:
`gpractice -help`
